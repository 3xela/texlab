Q1: Suppose that there exists two identity elements, $e,e^\prime$. Then for any element $g\in G$, we have that 
$$ge = ge^\prime = g$$
Multiplying by $g^{-1}$, we get that 
$$g^{-1}(ge) = (g^{-1}g)e = e = g^{-1}(ge^\prime) = (gg^{-1})e^\prime = e^\prime$$
We conclude that $e=e^\prime$. Hence the identity element is unique.
