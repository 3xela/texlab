Q8: If $G$ is abelian, then we have that $G^\prime = e$ since we can commute elements of the form $$ghg^{-1}h^{-1} = gg^{-1}hh^{-1}=e$$
Therefore $G/ G^\prime = G / \{e\}$. If we define $\phi:G \to G$ by $\phi(g)=g$, the kernel is clearly $e$. Hence by the first isomorphism theorem there is an isomorphism $\tilde{\phi}: G / \{e\} \to G$ which sends elements of the form $g\{e\}$ to $g$. Hence if $G$ is abelian we have that $G \cong G/ G^\prime$
