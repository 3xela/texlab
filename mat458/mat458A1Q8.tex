Q8: It is sufficient to show that for $\{(x_n,y_m)\} \to (0,0)$ then $B(x_n,y_m )\to 0$. Consider the mapping $B_x(y) = B(x,y)$. 
We know that there must exist some $C_x$ that satisfies $ \norm{B_x(y)} \leq C_x \norm{y}.$ By the uniform boundedness principle, we have that there is a maximal $C$
such that $\norm{B_x(y)} \leq C \norm{y}$ for all $x,y$. Therefore we have that $\norm{B(x_n,y_m)} \leq C \norm{B_{x_n}(y_m)} \to 0$ as $y_m,x_n\to 0$. Therefore $B(x,y)$ is continuous.
