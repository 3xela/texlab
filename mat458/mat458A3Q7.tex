\begin{document} \noindent 5.6.67: 
Claim: $|S_nx - Px| \to 0$ as $n\to \infty$. First if $x\in M$, then $$S_nx = \frac{1}{n} \sum_{i=0}^{n-1} U^{i}x = \frac{1}{n}\sum_{i=0}^{n-1}x = x$$ and $Px = x$. Now, if $x= y - Uy$, 
We see that $$\norm{S_n(x)} = \frac{1}{n}\norm{\sum_{i=0}^{n-1} U^i y - \sum_{i=0}^{n-1}U^{i+1}y} = \frac{1}{n} \norm{z - U^n z} \leq \frac{1}{n}\norm{z} + \norm{Uz} = \frac{2\norm{z}}{n}.$$
Thus we can make $\norm{S_n(x)}$ as small as we wish if $x$ is of the above form.
