\noindent Q3: Let $A$ be a bounded set. Since $A\subset \overline{A}$, we have that $J^\ast(A)\leq J^\ast(\overline{A})$ by A6Q4. Similary by A6Q4 it is immaterial whether we cover our desired set with open or closed rectangles. Hence if we cover $A$ with closed rectangles $\{I_k\}$, we will have that $\{I_k\}$ covers $A$ and $\overline{A}$. Since this is true for any closed cover we will have that $J^\ast(\overline{A})\leq J^\ast(A)$. Hence we conclude that $J^\ast(A) = J^\ast(\overline{A})$. Furthermore, since $\overline{A}$ is compact, we have that by A6Q4 $J^\ast(\overline{A}) = m^\ast(\overline{A})$.
