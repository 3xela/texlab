\noindent Q1: Let $A\subset \R^n$ be closed. Define $G_k = \{x:d(x,A)<\frac{1}{k}\}$. We have that each $G_k$ is open, and $A = \bigcap_{k\geq 1} G_k$. It follows immediately that every open set is an $F_\sigma$ set, since any open set is the compliment of some closed set. Taking the compliments in $\R^n$, we write our open set as the union of closed sets.
